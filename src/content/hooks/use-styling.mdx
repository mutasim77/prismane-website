# useStyling

useStyling hook provides a simple way to handle a click out of the element's constraints.

## Import

```js
import { useStyling } from "@prismane/core/hooks";
```

## Usage

```jsx
function Demo() {
  const [padding, setPadding] = useState("20px");

  setTimeout(() => {
    setPadding("40px");
  }, 2000);

  const styles = useStyling({
    padding: padding,
    width: "100px",
    height: "120px",
    backgroundColor: "#1e1e1e",
  });

  return <Box className={`${styles.join(" ")}`} />;
}
```

## API

### Parameters

| Name    | Type  | Description                                         | Default Value |
| ------- | ----- | --------------------------------------------------- | ------------- |
| `props` | `any` | The object that contains all the single properties. | -             |

### Return Value

| Name     | Type       | Description                                            |
| -------- | ---------- | ------------------------------------------------------ |
| `styles` | `string[]` | The array of classNames for each of the passed styles. |
