# Badge

Badge displays a small badge with short information to it's children.

## Import

```js
import { Badge } from "@prismane/core";
```

## Usage

By default, `Badge` uses the `primary` color.

```jsx
<Badge label="4">
  <Square size={fr(10)} bg="base" />
</Badge>
```
