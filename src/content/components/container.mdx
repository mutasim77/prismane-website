# Container

Container displays a container that constrains the content, whilst still remaining fluid.

## Import

```js
import { Container } from "@prismane/core";
```

## Usage

By default, the `Container` component does not have a set `maxSize`.

```jsx
<Container>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor
  incididunt ut labore et dolore magna aliqua. Urna porttitor rhoncus dolor
  purus non enim praesent elementum. Tortor vitae purus faucibus ornare
  suspendisse sed nisi.
</Container>
```

### Container With maxSize

When provided a `maxSize`, the container will restrain it's width to that size, but will always remain fluid.

```jsx
<Container maxSize="xs">
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor
  incididunt ut labore et dolore magna aliqua. Urna porttitor rhoncus dolor
  purus non enim praesent elementum. Tortor vitae purus faucibus ornare
  suspendisse sed nisi.
</Container>
```

### Container Sizes

```jsx
<Flex direction="column" gap={fr(1)} grow>
  <Container maxSize="xs" bg="red">
    Lorem ipsum dolor sit amet.
  </Container>
  <Container maxSize="sm" bg="green">
    Lorem ipsum dolor sit amet.
  </Container>
  <Container maxSize="base" bg="teal">
    Lorem ipsum dolor sit amet.
  </Container>
  <Container maxSize="md" bg="copper">
    Lorem ipsum dolor sit amet.
  </Container>
  <Container maxSize="lg" bg="ruby">
    Lorem ipsum dolor sit amet.
  </Container>
</Flex>
```
