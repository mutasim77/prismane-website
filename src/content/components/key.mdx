# Key

Key displays a container that is used to display keyboard key combinations.

## Import

```js
import { Key } from "@prismane/core";
```

## Usage

```jsx
<Key>Ctrl</Key>
```

### Key Combination

```jsx
<Flex gap={fr(1)} align="center">
  <Key>Shift</Key> + <Key>L</Key>
</Flex>
```
