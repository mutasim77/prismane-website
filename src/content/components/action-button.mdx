# ActionButton

ActionButton displays a square call-to-action button.

## Import

```js
import { ActionButton } from "@prismane/core";
```

## Usage

By default, `ActionButton` uses the `tertiary` variant.

```jsx
<ActionButton icon={<Heart />} />
```

### ActionButton Variants

```jsx
<>
  <ActionButton variant="primary" icon={<Heart />} />
  <ActionButton variant="secondary" icon={<Heart />} />
  <ActionButton variant="tertiary" icon={<Heart />} />
  <ActionButton variant="text" icon={<Heart />} />
</>
```

### ActionButton Sizes

```jsx
<>
  <ActionButton size="xs" icon={<Heart />} />
  <ActionButton size="sm" icon={<Heart />} />
  <ActionButton size="base" icon={<Heart />} />
  <ActionButton size="md" icon={<Heart />} />
  <ActionButton size="lg" icon={<Heart />} />
</>
```
