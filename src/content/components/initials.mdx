# Initials

Initials displays the initials of a provided name.

## Import

```js
import { Initials } from "@prismane/core";
```

## Usage

```jsx
<Initials name="Martin Petrov" />
```

### Avatar Example

```jsx
<Avatar>
  <Initials name="Martin Petrov" />
</Avatar>
```
