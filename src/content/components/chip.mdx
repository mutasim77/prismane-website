## Import

```js
import { Chip } from "@prismane/core";
```

## Usage

By default, `Chip` uses the `primary` color.

```jsx
<Chip>Default</Chip>
```

### Chip With Icon

```jsx
<Chip icon={<Fire />}>New Release</Chip>
```

### Chip Colors

By default, `Chip` uses the `primary` color. The `Chip` accepts all of [PrismaneColors](/docs/getting-started/globals#prismanecolors-type) as values of the `color` prop.

```jsx
<>
  <Chip>Primary</Chip>
  <Chip color="ruby">Ruby</Chip>
  <Chip color="lime">Lime</Chip>
  <Chip color="teal">Teal</Chip>
</>
```

### Chip Sizes

The `Chip` component supports all of the [PrismaneBreakpoints](/docs/getting-started/globals#prismanebreakpoints-type) as values for the `size` prop.

```jsx
<>
  <Chip size="xs">xs Size</Chip>
  <Chip size="sm">sm Size</Chip>
  <Chip size="base">base Size</Chip>
  <Chip size="md">md Size</Chip>
  <Chip size="lg">lg Size</Chip>
</>
```

### Text Colors

To change the text color of the `Chip`, you can pass a value to the `cl` prop.

```jsx
<Chip color="base" cl="diamond">
  Custom text color
</Chip>
```
