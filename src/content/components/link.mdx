## Import

```js
import { Link } from "@prismane/core";
```

## Usage

```jsx
<Link href="#">Click Me</Link>
```

### Link Underline Styles

```jsx
<>
  <Link href="#" underline="none">
    Click Me
  </Link>
  <Link href="#" underline="hover">
    Click Me
  </Link>
  <Link href="#" underline="always">
    Click Me
  </Link>
</>
```

### Foreign Link

Links that have the `foreign` prop will open in a new browser window.

```jsx
<Link href="https://www.google.com" foreign>
  Click Me
</Link>
```

### Before Function

`Link` exposes the support for a `before` function, which will execute, before the user will be redirected.

```jsx
<Link
  before={async () => {
    console.log("Redirecting...");
  }}
  href="https://www.google.com"
  foreign
>
  Click Me
</Link>
```

### Versatile Component

```jsx
<Link as="strong" href="https://www.google.com" foreign>
  Click Me
</Link>
```
