# Alert

Alert displays an important message for an action.

## Import

```js
import { Alert } from "@prismane/core";
```

## Usage

`Alert` component can display an alert for four kinds of messages: `info`, `success`, `warning` and `error`

```jsx
<Alert>Some demo alert!</Alert>
```

### Closable Alert

```jsx
<Alert closable>Some demo alert!</Alert>
```

### Custom Alert Icon

```jsx
<Alert icon={<Heart size={24} />} closable>
  Some demo alert!
</Alert>
```

### Complex Alert

```jsx
<Alert closable>
  <Alert.Title>Demo alert title</Alert.Title>
  <Alert.Description>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
    tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam,
    quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo
    consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse
    cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat
    non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
  </Alert.Description>
</Alert>
```

### Alert Variants

```jsx
<>
  <Alert variant="success">Some demo alert!</Alert>
  <Alert variant="info">Some demo alert!</Alert>
  <Alert variant="warning">Some demo alert!</Alert>
  <Alert variant="error">Some demo alert!</Alert>
</>
```
