# Checkbox

Checkbox displays a field that has a set of options, which can be filtered.

## Import

```js
import { Checkbox } from "@prismane/core";
```

## Usage

```jsx
function Demo() {
  const [value, setValue] = useState("");

  return (
    <Checkbox
      label="Default Field:"
      value={value}
      onChange={(e) => setValue(e.target.checked)}
    />
  );
}
```
