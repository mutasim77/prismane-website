# Accordion

Accordion displays a list of options that can be expanded or collapsed.

## Import

```js
import { Accordion } from "@prismane/core";
```

## Usage

You can have an expanded element by default, by providing a value to the `defaultValue` prop of the `Accordion` component.

```jsx
<Accordion defaultValue="first">
  <Accordion.Item value="first">
    <Accordion.Control>
      First
      <Accordion.Icon />
    </Accordion.Control>
```
