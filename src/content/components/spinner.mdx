# Spinner

Spinner displays a spinning container displaying a loading state.

## Import

```js
import { Spinner } from "@prismane/core";
```

## Usage

```jsx
<Spinner />
```

### Spinner Sizes

```jsx
<>
  <Spinner size="xs" />
  <Spinner size="sm" />
  <Spinner size="base" />
  <Spinner size="md" />
  <Spinner size="lg" />
</>
```
