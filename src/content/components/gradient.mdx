## Import

```js
import { Gradient } from "@prismane/core";
```

## Usage

By default, `Gradient` has the values of `from` - `["primary", 500]`, `to` - `["primary", 300]`, `deg` - `90`

```jsx
<Gradient w={360} h={120} />
```

### Gradient Colors

By providing values to the `from` and `to` props, you can change the start and end colors of the gradient.

```jsx
<Gradient w={360} h={120} from="teal" to="amethyst" />
```

### Gradient Degrees

By changing the value of the `deg` prop, you can change the degrees of the gradient.

```jsx
<Gradient w={360} h={120} from="ruby" to="lime" deg={45} />
```

### Versatile Component

```jsx
<Gradient
  as="a"
  href="https://www.google.com"
  target="_blank"
  w={360}
  h={120}
  from="copper"
  to="pink"
  deg={135}
/>
```
