<Versatile>Gradient</Versatile>

Gradient displays a container with gradient background.

## Import

```js
import { Divider } from "@prismane/core";
```

## Usage

By default, `Gradient` has the values of `from` - ["primary", 500], `to` - ["primary", 300], `deg` - 90

```jsx
<Gradient w={360} h={120} />
```

### Custom Gradient

```jsx
<Gradient w={360} h={120} from={["teal", 600]} to={["ruby", 600]} deg={45} />
```

### Versatile Component

```jsx
<Gradient
  as="a"
  href="https://www.google.com"
  target="_blank"
  w={360}
  h={120}
  from="copper"
  to="pink"
  deg={135}
/>
```
