<Versatile>Paper</Versatile>

Paper displays a container with background and shadow.

## Import

```js
import { Paper } from "@prismane/core";
```

## Usage

```jsx
<Paper w={120} h={120} />
```

### With Shadow

```jsx
<Paper w={120} h={120} shadow />
```

### Versatile Component

```jsx
<Paper
  as="a"
  href="https://www.google.com"
  target="_blank"
  w={120}
  h={120}
  shadow
/>
```
