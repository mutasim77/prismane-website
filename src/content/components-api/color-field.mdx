# ColorField

Color field displays a field that has a set of colors, which can be chosen.

## Import

```js
import { Drawer } from "@prismane/core";
```

## Usage

```jsx
function Demo() {
  const [color, setColor] = useState("");

  return (
    <ColorField
      placeholder="Default Field"
      label="Default Field:"
      value={color}
      onChange={(e) => setColor(e.target.value)}
    />
  );
}
```

### Custom Set Of Colors

`ColorField` accepts a parameter of `options`, in which you can specify what color the field should offer as options. They can either be PRISMANE colors or custom colors.

```jsx
function Demo() {
  const [color, setColor] = useState("");

  return (
    <ColorField
      options={["teal", "#1e1e1e"]}
      placeholder="Default Field"
      label="Default Field:"
      value={color}
      onChange={(e) => setColor(e.target.value)}
    />
  );
}
```
