# Popover

Popover displays a dialog container that is shown around a control.

## Import

```js
import { Popover } from "@prismane/core";
```

## Usage

```jsx
<Popover>
  <Popover.Control>
    <Button>Toggle Popover</Button>
  </Popover.Control>
  <Popover.Panel>This is the content of the popover!</Popover.Panel>
</Popover>
```
