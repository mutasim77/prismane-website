# Hidden

Hidden displays a container that hides it's content.

## Import

```js
import { Hidden } from "@prismane/core";
```

## Usage

```jsx
<Button>
  Hello <Hidden>World</Hidden>
</Button>
```

### Common Uses

The `Hidden` component is usually used to hide inputs in custom made input components

```jsx
<Box
  as="label"
  htmlFor="checkbox"
  px={fr(2)}
  py={fr(0.5)}
  bg="primary"
  br={2}
  cs="pointer"
>
  Click to check
  <Hidden>
    <input
      name="checkbox"
      id="checkbox"
      type="checkbox"
      onChange={(e) => {
        console.log("Checkbox is checked: ", e.target.checked);
      }}
    />
  </Hidden>
</Box>
```
